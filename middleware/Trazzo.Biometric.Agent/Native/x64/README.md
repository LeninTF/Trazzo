# SDK nativo ZKTeco

Copiar aquí el archivo oficial x64 `libzkfpcsharp.dll` del SDK ZKTeco.

Ruta esperada:

```text
Native/x64/libzkfpcsharp.dll
```

No incluyas la DLL en el repositorio salvo que tu licencia con ZKTeco lo permita explícitamente.

Si falta esta DLL, la captura con hardware no puede ejecutarse. El proyecto compila y el agente informa:

```text
No se encontró la DLL del SDK ZKTeco o no se pudo cargar.
```
